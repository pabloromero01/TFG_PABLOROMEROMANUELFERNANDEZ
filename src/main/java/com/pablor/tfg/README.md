# TFG - App de Comparación de Estadísticas de Fútbol

## Descripción
Aplicación para comparar estadísticas de fútbol entre jugadores y equipos (goles, asistencias, minutos, tarjetas, rendimiento por temporada, etc.), mostrando comparativas y métricas relevantes para análisis deportivo.
